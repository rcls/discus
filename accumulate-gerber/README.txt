The board is rectangular 100mm by 50mm.  The gerber files are named as per
the instructions on the pcb-specification.com website:

TopSide.gbr
TopSilkscreen.gbr
TopSolderMask.gbr

BotSide.gbr
BotSilkscreen.gbr
BotSoldermask.gbr

BoardOutline.gbr

There are two drill files:

Drill.cnc : Plated holes.
UnplatedDrill.cnc : Unplated holes.
